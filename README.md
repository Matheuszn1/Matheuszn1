## 📌 Projetos em destaque

### 🛒 Gerenciador de Produtos de E-commerce em C++

Sistema acadêmico desenvolvido em C++ para gerenciamento de produtos de e-commerce, com cadastro, edição, filtros, busca, ordenação, ranking e persistência em arquivo binário.

**Recursos utilizados:**

- Cadastro e listagem de produtos
- Controle de estoque
- Busca por nome
- Filtro por categoria e faixa de preço
- Ordenação por preço e avaliação
- Árvore AVL para indexação
- Busca binária
- QuickSort
- Arquivos binários
